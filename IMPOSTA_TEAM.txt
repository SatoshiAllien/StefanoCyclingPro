FIRMA CODICE — OBBLIGATORIO per iPhone 16 Pro + Apple Watch
=============================================================

ERRORE: "Signing requires a development team"

SOLUZIONE (scegli una):

A) File xcconfig (consigliato):
   1. Apri Config/Development.xcconfig
   2. Inserisci: DEVELOPMENT_TEAM = IL_TUO_TEAM_ID
   3. Clean (⇧⌘K) → Build (⌘B)

B) In Xcode:
   1. Target StefanoCyclingPro → Signing → Team → seleziona il tuo account
   2. Target StefanoCyclingProWatch → Signing → Team → stesso account
   3. Clean (⇧⌘K) → Build (⌘B)

Team ID: https://developer.apple.com/account → Membership

Workflow invariato:
   git pull origin main
   open StefanoCyclingPro.xcodeproj
