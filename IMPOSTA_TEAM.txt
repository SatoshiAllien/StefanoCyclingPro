FIRMA CODICE — iPhone + Apple Watch
====================================

1. Apri Config/Development.xcconfig
2. Sostituisci TEAMID_PLACEHOLDER con il tuo Team ID (10 caratteri)
3. Xcode → entrambi i target → Signing → Automatically manage signing ON
4. Clean (⇧⌘K) → Build (⌘B) → Run su iPhone (⌘R)

Bundle IDs:
  iOS:     com.example.StefanoCyclingPro
  watchOS: com.example.StefanoCyclingPro.watch

Workflow:
  git clone https://github.com/SatoshiAllien/StefanoCyclingPro.git
  cd StefanoCyclingPro
  open StefanoCyclingPro.xcodeproj
